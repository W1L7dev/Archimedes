Archimedes is a collection of education-related CLI and TUI tools mainly for Arch Linux. All the TUI apps are based on ncurses and a custom framework.

## Installation

For now, the only way of using Archimedes tools is to build them from the source code, since the development is still in an early stage. We consider putting the tools on the Arch AUR, and using docker for cross-plateform compatibility.

> [!WARNING]
> As mentionned, the development is in a very early state, meaning that the code might be unstable, leading to unwanted damage to your machine.

## Running

As of right now, there isn't a way of running the tools.

## Tools & Features

As of right now, no features have been implemented yet. Here's a preview of wanted features.

> [!NOTE]
> If you have an idea for a feature to implent, open an issue with the tag `feature`.

| Name            | Description                                                                                                     | Command      |
| --------------- | --------------------------------------------------------------------------------------------------------------- | ------------ |
| Archimedes      | A parent TUI app for managing the suite.                                                                        | `archimedes` |
| MathType        | A TUI app for math stuff. Supports LaTeX. Scientific calculator and graphing calculator integrated              | `mtype`      |
| ChemType        | A TUI app for chemistry stuff. Supports LaTeX. Can auto-balance chemical equations. Interactive periodic table. | `ctype`      |
| Balancer        | A standalone command to balance chemical equations.                                                             | `balancer`   |
| Data Visualizer | A standalone command to visualize data. Supports table view and tree view.                                      | `datavis`    |
